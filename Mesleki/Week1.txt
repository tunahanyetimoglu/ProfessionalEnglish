What is the system unit?

 - Case that contains electronic components of the computer used to process data.
  
 - Verileri işlemek için kullanılan bilgisayarın elektronik bileşenlerini içeren durum.

What are common(ortak) components(bileşenler) inside the system unit?

-Processor            ->    İşlemci
-Memory               ->    Bellek
-Adapter cards;       ->    Adapter Kartları;
      -Sound card     ->    Ses Kartı
      -Video card     ->    Video Kartı
-Ports                ->    Portlar
-Drive bays           ->    Drive Koyları
-Power supply         ->    Güç Kaynağı

What is the motherboard(anakart)?

-Main circuit board in system unit  ->  (Sistem ünitesindeki ana devre kartı)

-Contains adapter cards,processor,chips,and memory modules  ->  (Bağdaştırıcı kartları, işlemci, cips ve bellek modülleri içerir)

What is chip?

-Small piece of semi-conducting material on which integrated circuits are etched
    *Integarated circuits contain many microscopic pathways capable of carrying electrical current
-Chips are packaged so they can be attached to a  circuit board.

What is the central processing unit(CPU)?

- Interprets and carries out basic instructions that operate a computer 

( Bir bilgisayarı çalıştıran temel talimatları yorumlar ve yürütür.)

  *Control unit directs and coordinates operations in computer .( Kontrol birimi bilgisayardaki işlemleri yönlendirir ve koordine eder)

  *Arithmetic logic unit(ALU) performs arithmetic, comparison,and logical operations 

  (Aritmetik mantık birimi (ALU), aritmetik, karşılaştırma ve mantıksal işlemleri gerçekleştirir.)

-Also called the processor

What is a machine cycle?

-Four operations of the CPU comprise a machine cycle

Step 1. Fetch (Getir)

    Obtain program instruction or data item from memory ( Bellekten program talimatı veya veri öğesi edinin )

Step 2. Decode  ( Kod Çözme )

	Translate instruction into commands (Talimatları komutlara çevir)

Step 3. Execute  (Yürüt)

	Carry out command ( Emir verme)

Step 4. Store (Mağaza)

	Write result to memory ( Sonucu hafızaya yaz)
